# ccpkolkata

Live Preview: <https://ccpkolkata.asimbera.tech>

## Building

- Prerequiste: Node 14.17.3 or later, Yarn package manager
- Run `yarn` to install required dependencies.
- Run `yarn start` or `yarn build` to run or build.
